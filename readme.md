# app-path-cli

> Get the path to an app (macOS)

## Install

```
$ npm install --global app-path-cli
```

## Usage

```
$ app-path --help

  Usage
    $ app-path <app-name|bundle-id>

  Examples
    $ app-path Safari
    /Applications/Safari.app
    $ app-path com.apple.Safari
    /Applications/Safari.app
```
